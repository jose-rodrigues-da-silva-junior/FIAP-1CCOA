# 🎓 FIAP - Ciência da Computação (1CCOA)

Este repositório serve como documentação da minha jornada acadêmica na **FIAP**. Todo o aprendizado é estruturado através da metodologia **PBL (Project-Based Learning)**, onde o conhecimento é aplicado diretamente na resolução de desafios reais.

## 📂 Organização por Fases
O ano letivo é dividido em fases, onde cada uma contém:

* **The Challenge:** O problema central a ser resolvido.
* **Integrated Modules:** Disciplinas que fornecem o arsenal técnico para o projeto.
* **Knowledge Assessment:** Registros de Fast Tests e Quizzes de performance.
* **Boosters:** Cursos complementares e certificações que apoiam a entrega da fase.
